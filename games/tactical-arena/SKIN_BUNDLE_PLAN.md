# Skin Bundle Plan

This is a merchandising plan for the current skin inventory. The game can keep
all skins unlocked while testing, but these groupings give us a future store
shape where some skins are permanent, some rotate seasonally, and some stay rare.

Inventory baseline: 112 authored skins across 22 units.

## Availability Types

| Type | Meaning | Suggested use |
| --- | --- | --- |
| Free | Always available without purchase. | Classic/base unit looks. |
| Starter | Cheap or launch-friendly collection. | Broad packs that teach players skins exist. |
| Permanent | Always in the store. | Core fantasy, role, or small theme packs. |
| Seasonal | Returns on a calendar/event schedule. | Summer, Halloween, Blood Moon, holidays. |
| Limited | Rotates rarely and should feel special. | Event reward reruns, anniversary drops, premium chase skins. |
| Single | Sold alone, not only inside a pack. | Hero skins, unit-defining skins, unusually specific skins. |
| Promo | Tied to community, creator, achievement, or giveaway. | Discord/community skins and non-store rewards. |

## Pack Details

### Summer Vibes

Availability: seasonal summer. Rarity: rare seasonal mega pack.

Includes all 22 units:

Swordsman, Archer, Mystic, Magician, Paladin, Necromancer, Ghoul, Sniper,
Witch Doctor, Father Time, Juggernaut, King, Angel, Monk, Gargoyle, Nemesis,
Virus, Clod, Fat Knight, Fat Wizard, Fat Cleric, Fat Bowman.

Recommendation: keep unlocked during testing. Later, make it a summer rotation
pack and optionally give early testers permanent ownership as a thank-you.

### Arcane Order

Availability: permanent store or long evergreen rotation. Rarity: uncommon/core.

Includes 19 units:

Swordsman, Archer, Mystic, Magician, Necromancer, Ghoul, Sniper, Witch Doctor,
Father Time, Juggernaut, King, Gargoyle, Nemesis, Virus, Clod, Fat Knight,
Fat Wizard, Fat Cleric, Fat Bowman.

Recommendation: this is the safest always-available paid pack. It covers most
of the roster and has broad fantasy appeal without feeling holiday-locked.

### Blood Moon

Availability: seasonal event. Rarity: epic limited.

Includes 10 units:

Swordsman, Archer, Mystic, Magician, Paladin, Necromancer, Ghoul, Sniper,
Witch Doctor, Monk.

Recommendation: treat Blood Moon as a high-value event. Rotate it for short
windows, bundle the full set, and allow a few individual purchases for favorite
units. Avoid keeping it permanently available.

## Future Implementation Notes

- Keep `src/ui/skinManifest.generated.js` generated from files only. Do not put
  pricing, rarity, or availability in the generated manifest.
- Add a separate authored catalog later, for example `src/ui/skinCatalog.js`,
  with pack ids, rarity, availability, and exact `{ type, slug }` skin refs.
- Skin ownership should resolve after the manifest loads. Test/dev mode can keep
  forcing everything unlocked while production checks player entitlements.
- Packs should reference exact unit plus slug pairs, not slug alone. Some slugs
  are broad collections, while others appear on only one unit.
- Keep Classic/base looks free forever.
- Prefer rotating old limited packs back in occasionally instead of making them
  impossible to get forever.
