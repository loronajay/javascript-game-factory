# Build Buddy

`build-buddy/` is currently a design-only folder for a two-player online cooperative platformer.

## Current status

- No cabinet code is implemented here yet.
- `BUILD_BUDDY_GDD.md` is the source of truth for the concept, match structure, and platform fit.

## Concept snapshot

- Two players share a 10-stage run.
- One player is the Runner and the other is the Builder.
- Roles swap every stage.
- The Runner handles momentum platforming while the Builder edits support tools in real time.

When implementation starts, keep the full game self-contained in this folder and follow the repo game structure conventions described in `games/README.md`.
