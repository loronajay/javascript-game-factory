# Questionable Decisions Mini-Games

This folder holds the penalty mini-game concepts for Questionable Decisions.

## Included concepts

- `bomb-diffuser/`
- `brainscrambler/`
- `cabinet-says/`
- `memory-sketch/`
- `pattern-panic/`
- `pressure-typer/`
- `stack-overflow/`
- `through-the-fire/`

Each folder currently contains a `GDD.md` for that penalty mode.

## Purpose

These are design specs, not isolated production games. Their job is to define failure moments that the main Questionable Decisions cabinet can call into after an incorrect answer.
