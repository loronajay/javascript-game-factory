# Changelog

Dated history extracted from the root `CLAUDE.md` (2026-07-23) so that file can stay a lean orientation guide instead of an ever-growing log. This file is a curated narrative, not a replacement for `git log` — read it for *why*/*what shipped when*, not for line-level diffs.

## Factory Launcher — Security Hardening (2026-07)

A hardening pass closed the money-critical holes. Backend (`platform-api/`): the public game-progress `/claims` route now rejects `premium-*` kinds (premium entitlements are Stripe-fulfillment-only); CORS is restricted to an allow-list (`https://loronajay.github.io` + localhost, extendable via `ALLOWED_ORIGINS`) instead of reflecting any origin with credentials; `/activity` requires auth and stamps the server-verified actor; JWT verification is pinned to HS256; per-IP rate limiting guards auth + Stripe checkout-session creation; uploads are validated by magic bytes (not the client MIME), which also guards the Cloudinary `raw` audio store. Tactical Arena's economy is now server-authoritative for signed-in players — Valor is spent through a server endpoint (atomic deduct+grant), and ownership is reconciled down to the server's set on boot (injected local ownership is dropped and self-heals) with a one-time backfill so no existing progress is lost. Full model + invariants in `games/tactical-arena/SECURITY.md`.

## Game-Level Feature History

Online is one unified lobby-based mode (1v1 = a 2-player lobby): deterministic lockstep over the server's generic v2 lobby relay, owner = state-hash authority, board/format/squads exchanged in-band, a dropped player is conceded by the remaining owner; the server change is just a config-only `lobbyGame` (limits 2–4) — no match engine. v1 code-complete + headless-validated (121/121) + real-server lobby smoke, not yet multi-tab hand-playtested. Local hot-seat 1v1 duel, single-player vs CPU (Easy/Normal/Hard), and online 1v1 (a 2-player lobby). Roster is the full legacy drafted set (Swordsman, Archer, Mystic, Magician, Paladin, Necromancer, Sniper, summon-only Ghoul) plus a growing set of rebuild-original units (Witch Doctor, Father Time, Juggernaut, King, Monk, Angel, Gargoyle, Nemesis, Virus, Clod, the four-unit "fat squad", Miner, Riot Cop, Big Brother/Little Brother, Blacksword, Ronin, Mother Nature, Summoner, Treant). Online Versus shipped: blind-pick, Quick Match + private code, deterministic lockstep, code-complete + headless-validated + real-server two-client relay smoke, not yet multi-tab hand-playtested. Board tokens use painted sprites with carved-SVG fallback.
